# Hello World!

Welcome to my first blog post! This is a simple example of how to create content using Fumadocs with TanStack Start.

## Getting Started

This blog post demonstrates various Fumadocs features:

### Basic Markdown

You can use all standard Markdown features:
- **Bold text**
- *Italic text*
- `inline code`
- Links to [external sites](https://fumadocs.dev)

### Code Examples

Here's a simple JavaScript example:

```javascript
function sayHello(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to my blog, ${name}`;
}

sayHello('World');
```

And here's some TypeScript:

```typescript
interface BlogPost {
  title: string;
  description: string;
  publishedAt: Date;
  tags?: string[];
}

const firstPost: BlogPost = {
  title: "Hello World",
  description: "My first blog post",
  publishedAt: new Date()
};
```
